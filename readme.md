## PHP + Laravel + Rest + Swagger

###Installation

	$ git clone https://github.com/go-labs/php_rest_poc
	$ cd php_rest_poc
	$ composer update
	$ cp .env.example .env
	$ edit .env file with database information
	$ php artisan migrate
	$ php artisan serve
	$ open http://localhost:8000/api-docs in your favorite browser
