## PHP + Laravel + Rest + Swagger

###Installation

	$ git clone git@github.com:go-labs/php_rest_poc.git
	$ cd php_rest_poc
	$ composer install
	$ cp .env.example .env
	$ edit .env file with database information
	$ php artisan migrate
	$ php artisan serve
	$ open http://localhost:8000 in your favorite browser
